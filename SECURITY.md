# Security Policy

## Supported Versions

The latest published `0.x` release line is supported for security fixes.

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Preferred reporting path:

1. Open a private GitHub security advisory for this repository:
   `https://github.com/HoltzTomas/s3-files-ai-sdk/security/advisories/new`
2. Include the affected version, deployment mode (`direct` or `remote`), and reproduction details
3. If the issue relates to remote mode, include whether bearer auth, proxy routing, or path isolation is involved

We will aim to acknowledge new reports within 5 business days and will coordinate remediation and disclosure with the reporter.
